A "Council of Experts" agent card — titled icon header, italic analysis paragraph, and a stack of recommendation chips in inset wells. Use in a 3-up grid.

```jsx
<ExpertCard
  name="Cardio Guardian" icon={<i data-lucide="activity" />}
  accent="var(--color-primary)"
  analysis="HRV shows high parasympathetic activity…"
  recommendations={["Vigorous workout recommended", "Focus on explosive power"]}
/>
```

Props: `name`, `role`, `icon`, `accent`, `analysis`, `recommendations[]`.
