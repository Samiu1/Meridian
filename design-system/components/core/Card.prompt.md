The foundational neumorphic pebble surface — compose every panel, tile, and section from it.

```jsx
<Card elevation="flat" radius="3xl" padding="lg">…</Card>
<Card elevation="inset" radius="xl">pressed well</Card>
```

Elevations: `flat` (raised), `inset` (pressed), `sm`, `float`, `none`. Radii: `md`→`4xl` (bigger = hero/section). Padding presets: `none`,`sm`,`md`,`lg`,`xl`. Render any tag via `as`.
