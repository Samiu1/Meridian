A neumorphic inset text field — the well is pressed into the paper, with an optional leading icon. Use for login and settings forms.

```jsx
<Input icon={<i data-lucide="mail" />} placeholder="Garmin Email" type="email" />
<Input placeholder="MFA Code" align="center" />
```

Props: `icon`, `align` ("left" | "center"), `type`, `value`, `onChange`, `disabled`, `ariaLabel`.
